Buddyships

Create

Adds one or more users to the list of current user's buddies.
Requires Authentication

POST /v1/users/me/buddyships
Example

Request body:

{
  "buddyships": [
    {
      "facebook_id": "100002734755203"
    },
    {
      "facebook_id": "100002950674984"
    }
  ]
}
