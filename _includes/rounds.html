Create

Starts a new round.
Requires Authentication

POST /v1/rounds

Parameters:

Parameter Description
game_name The round's game. It can be among these:
  • dota2
  • go
  • table_football
arena_properties Dictionary containing the properties of the arena which can be either the foursquare_id or the location coordinates: 
{ foursquare_id": "5104" }
{ "latitude": 30, "longitude": 50 }
participation_list Array of dictionaries containing the "team" and "user" properties.
  • "team" is used to decide which team the user will join; if none is provided it will be automatically assigned to the first one avaiable. See the description of games to know which teams you can use.
  • "user" can contain the Playround id of the user or the facebook_id of a unregistered user, who will be imported into Playround.
Example

Request body:

{
  "round": {
    "game_name": "dota2",
    "arena_properties": { "foursquare_id": "5104" },
    "participation_list": [
      {
        "team": "dire",
        "user": { "id": "c0731481-35d0-47c1-ab84-c0c7b0bef10c" }
      },
      { "user": { "facebook_id": "100002734755203"} }
    ]
  }
}

Invalid attributes

Posting invalid attributes will result in a response with status 422 and a body containing the errors relative to invalid fields.

Example

Request body:

{
  "round": {
    "game_name": "lol"
  }
}

Response:

HTTP/1.1 422 Unprocessable Entity

{
  "errors": {
    "game_id": [
      "can't be blank"
    ],
    "arena_id": [
      "can't be blank"
    ],
    "base": [
      "game_name must be among: dota2, table_football, go"
    ]
  }
}
