Participations

Players can use this endpoint to manage their participation to rounds. They can either join a match (after invitation or not) or leave it.

Create

The current user joins the round.
Requires Authentication

POST /v1/rounds/:round_id/participations

Parameters:

Parameter Description
team Check out the game section to know which teams you can use
Example

Request body:

{
  "participation": {
    "team": "radiant"
  }
}

Requires Authentication
