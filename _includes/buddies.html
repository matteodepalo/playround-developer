Create

Adds a user to the list of current user's buddies.
Requires Authentication

POST /v1/users/me/buddies
Example

Request body:

{
  "buddies": [
    {
      "facebook_id": "100002734755203"
    },
    {
      "facebook_id": "100002950674984"
    }
  ]
}
