Winnings

Create

Determines the winning team of the round. This action can be perfomed only on ongoing rounds. Requires Authentication and Authorization
Only the creator of the round can determine the winner.

POST /v1/rounds/:round_id/winnings
Example

Request body:

{
  "winning": {
    "team_name": "radiant"
  }
}

Response:

HTTP/1.1 200 Success
