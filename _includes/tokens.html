Tokens

Create

Creates an api token from a Facebook access token. You can then use the api token returned to authenticate, passing it like described above in the basics authentication section

POST /v1/tokens

Parameters

Parameter Description
token A dictionary containing the Facebook access token 
{ "facebook_access_token": FB_TOKEN }

{
